Dynamically-typed languages

Dynamic typing is a feature of JavaScript that allows you to assign a value of any type to a variable without declaring its type explicitly. This means that you can change the type of a variable at runtime by assigning a new value of a different type.

Learn more about the Javascript`s dynamically - type feature

// This demonstrates the dynamic typing feature of JavaScript, where a variable can be assigned a value of any type without explicitly declaring its type, and the type of the variable can change at runtime.

let myVariable = 42; // myVariable is a number

console. log(typeof myVariable); // Output: "number"

myVariable = "Hello World!"; // myVariable is now a string

console. log(typeof myVariable); // Output: "string"

myVariable = true; // myVariable is now a boolean

console. log(typeof myVariable); // Output: "boolean"

HISTORY OF JAVASCRIPT

Creation of Javascript

JavaScript was created by Brendan Eich, a software developer at Netscape Communications, in just 10 days. It was originally called Mocha, then renamed LiveScript, and finally renamed JavaScript to capitalize on the popularity of Sun Microsystems' Java programming language.

1995

"JavaScript is the world's most misunderstood programming language."

- Brendan Eich

1996

Microsoft Adoption

JScript is a scripting language that was developed by Microsoft as a variant of the JavaScript language. It was first introduced in 1996 and was primarily used for client-side scripting in web development, particularly for Internet Explorer.

ECMA standardization

JavaScript was submitted to Ecma International, a standards organization for information and communication systems, for standardization. The first edition of the ECMAScript standard was released, which specified the core features of the language.

1996
2005

Introduction of AJAX

JavaScript AJAX (Asynchronous JavaScript and XML) is a technique that allows web pages to update content dynamically without the need for a page refresh. This is achieved by using a combination of JavaScript and other web technologies like XML, HTML, and CSS to create asynchronous requests to a web server.

Javascript on servers

Node.js is a JavaScript runtime environment that allows developers to use JavaScript for both front-end and back-end development. It is fast, scalable, and has a large ecosystem of third-party packages available via npm.

2009
2015

Introduction of ES6

ES6 (ECMAScript 2015) is a major update to JavaScript that introduced new features and improvements, such as arrow functions, classes, template literals, let and const variables, spread and rest operators, and destructuring. It has made JavaScript more powerful, flexible, and easier to use and understand, and many of its features have become widely adopted in modern JavaScript development.
